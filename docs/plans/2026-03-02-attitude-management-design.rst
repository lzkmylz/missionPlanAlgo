卫星姿态管理系统设计
====================

**日期**: 2026-03-02
**作者**: Claude Code
**状态**: 待实现

概述
----

背景
~~~~

当前调度系统在规划成像任务时，未考虑成像完成后卫星姿态的复位问题。现实卫星在执行成像任务后，通常需要切换回对日定向姿态以最大化太阳帆板发电效率，为后续任务充电。

目标
~~~~

设计并实现一个姿态管理系统，使调度器能够在以下姿态模式间智能切换：

- 成像姿态（侧摆/斜视）
- 对地定向姿态（星下点）
- 对日定向姿态（充电）
- 数传姿态（对地面站）
- 实传姿态（边成像边数传）
- 动量卸载姿态（维护模式）

核心原则
~~~~~~~~

1. **空闲期间自动对日**：当任务间隔超过阈值时，自动切换对日定向姿态充电
2. **电量驱动决策**：电量低于阈值时，强制进入对日定向
3. **动量卸载优先**：定期强制插入动量卸载，确保系统稳定
4. **基于角度计算机动时间**：使用Orekit计算太阳位置，动态计算机动角度和时间
5. **余弦衰减功率模型**：根据姿态偏离对日方向的角度，计算发电功率衰减

------------

姿态模式详细定义
~~~~~~~~~~~~~~~~

+---------------+-------------------+----------------------------+----------+----------+------------------+----------+
| 模式          | 英文名称          | 定义                       | 滚转角   | 俯仰角   | 发电效率         | 优先级   |
+===============+===================+============================+==========+==========+==================+==========+
| 对日定向      | Sun Pointing      | 太阳帆板法向对准太阳       | 动态计算 | 动态计算 | 95-100%          | 低       |

+---------------+-------------------+----------------------------+----------+----------+------------------+----------+
| 数传姿态      | Downlinking       | 指向地面站                 | 地面站决 | 地面站决 | 余弦衰减         | 高       |
+---------------+-------------------+----------------------------+----------+----------+------------------+----------+

姿态切换路径与机动时间
~~~~~~~~~~~~~~~~~~~~~~

+---------------------------+-------------------------+----------+
| 切换路径                  | 机动时间计算方式        | 复杂度   |
+===========================+=========================+==========+
| 对日定向 ↔ 对地定向       | 基于太阳-卫星-地心夹角  | 中       |
+---------------------------+-------------------------+----------+
| 对日定向 ↔ 成像姿态       | 基于太阳-卫星-目标夹角  | 高       |
+---------------------------+-------------------------+----------+
| 对日定向 ↔ 数传姿态       | 基于太阳-卫星-地面站    | 高       |
+---------------------------+-------------------------+----------+
| 对地定向 ↔ 成像姿态       | 基于滚转/俯仰角变化     | 低       |
+---------------------------+-------------------------+----------+
| 对地定向 ↔ 数传姿态       | 基于地面站方位角        | 中       |
+---------------------------+-------------------------+----------+
| 成像姿态 ↔ 数传姿态       | 目标与地面的夹角        | 中       |
+---------------------------+-------------------------+----------+
| 任意 → 实传姿态           | 需满足成像+数传约束     | 高       |
+---------------------------+-------------------------+----------+
| 任意 → 动量卸载           | 通常为切换到对日定向    | 中       |
+---------------------------+-------------------------+----------+

机动时间公式::

    slew_time = max(|Δroll|, |Δpitch|) / max_slew_rate + settling_time

发电功率模型
------------

余弦衰减模型
~~~~~~~~~~~~

当卫星不在对日定向姿态时，太阳帆板与太阳光线的夹角导致发电效率下降：

功率计算公式::

    P_generation = P_max × cos(θ_sun_panel) × eclipse_factor

其中：
- P_max：帆板正对太阳时的最大功率（W）
- θ_sun_panel：太阳矢量与帆板法向的夹角（度）
- eclipse_factor：地影因子（地影期为0，日照期为1）

关键接口
~~~~~~~~

姿态模式枚举::

    class AttitudeMode(Enum):
        SUN_POINTING = "sun_pointing"      # 对日定向
        NADIR_POINTING = "nadir_pointing"  # 对地定向
        IMAGING = "imaging"                # 成像姿态
        DOWNLINK = "downlink"              # 数传姿态
        REALTIME = "realtime"              # 实传姿态
        MOMENTUM_DUMP = "momentum_dump"    # 动量卸载

姿态切换请求::

    @dataclass
    class AttitudeTransition:
        from_mode: AttitudeMode
        to_mode: AttitudeMode
        timestamp: datetime
        satellite_position: Tuple[float, float, float]
        sun_position: Optional[Tuple[float, float, float]] = None
        target_position: Optional[Tuple[float, float]] = None
        ground_station_position: Optional[Tuple[float, float]] = None

姿态切换结果::

    @dataclass
    class TransitionResult:
        slew_time: float           # 机动时间（秒）
        slew_angle: float          # 机动角度（度）
        roll_angle: float          # 滚转角变化（度）
        pitch_angle: float         # 俯仰角变化（度）
        power_generation: float    # 发电功率（W）
        feasible: bool
        reason: Optional[str] = None

配置参数
~~~~~~~~

::

    @dataclass
    class AttitudeManagementConfig:
        """姿态管理系统配置"""

        # 空闲时间阈值（秒），超过此值切换到对日定向
        idle_time_threshold: float = 300.0  # 5分钟

        # 电量阈值，低于此值强制对日定向
        soc_threshold: float = 0.30  # 30%

        # 动量卸载间隔（秒）
        momentum_dump_interval: float = 14400.0  # 4小时

        # 动量卸载持续时间（秒）
        momentum_dump_duration: float = 600.0  # 10分钟

        # 机动稳定时间（秒）
        settling_time: float = 5.0

        # 最大机动角速度（度/秒）
        max_slew_rate: float = 3.0

        # 是否启用对日定向优化
        enable_sun_pointing_optimization: bool = True

        # 是否启用动量卸载
        enable_momentum_dumping: bool = True
